# 상명대학교 천안캠퍼스 2023 캡스톤 경진대회
## 프로젝트 기간
2023.02 - 2023.10 

## 개요
헬스케어 임베디드 시스템으로 사용자의 건강 상태를 모니터링하고 데이터를 수집, 분석하여 사용자에게 필요한 건강 정보를 제공한다.
카메라를 통해 얼굴, 어깨의 불균형을 측정하고 기록된 결과를 통해 건강상태를 확인할 수 있다.

## 주요 기능
- 얼굴 불균형 측정
- 어깨 불균형 측정
- 라이프로그 기록
- LDL 예측
- 수면기록 측정
